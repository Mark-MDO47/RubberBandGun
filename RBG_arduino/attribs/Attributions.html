The FOOF Rubber Band Gun

The original concept for shooting sounds for the Rubber Band Gun came from the LIGO "soundified" recordings of two neutron stars colliding. See below for info on where to find these sounds.

Another space sound is from the NASA Cassini mission representing radio emissions from Saturn: http://www-pw.physics.uiowa.edu/space-audio/cassini/SKR1/

Some sounds used on this SD card were recorded by me, some were generated by me using eSpeak, and some came from the sources below. Sounds may have been cut into pieces, altered, combined, etc.

Sounds on this SD card have a naming convention to optimize use with the YX5200 module. These names do not correspond directly to the filenames listed below, which are the source file names for some of the sounds.
